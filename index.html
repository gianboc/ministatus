How fresh is what you see?

  • Each machine reports every 5 minutes. The "heartbeat … ago" on a tile is the true age of that machine's numbers — it is the figure to trust.
  • GitHub adds its own small delay. The machines write their numbers to GitHub, and GitHub hands this page a copy it may have kept for up to ~5 minutes. So in the worst case what you see is 5–10 minutes behind reality — normal for this page, and fine for its purpose.
  • "checked …" in the header is only when this page last asked GitHub (it re-asks every 60 seconds). A page checked seconds ago can still be showing numbers a few minutes old — the two times measure different things.
  • What the badges mean: a heartbeat younger than ~12 min is normal (UP). If a machine goes silent but the other minis can still reach it, it's not reporting — the machine is fine, only its reporting broke. If its peers can't reach it either, it's unreachable — really down. Silence over 20 min with no peer information shows as likely down.
